# AGENTS.md

## Missione del repository
Costruire un gioco 2D intitolato **Foscolo: La religione delle illusioni**.

Il gioco deve tradurre in forma interattiva il pensiero di Foscolo, senza ridurlo a una spiegazione scolastica. Il centro del progetto è il rapporto tra reale meccanicistico, morte, memoria e religione delle illusioni.

## Obiettivi di stile
- stile visivo: retro anni Ottanta
- atmosfera: gotico-mediterranea
- tono: poetico, sobrio, malinconico
- evitare stile cartoon generico
- evitare eccesso di elementi arcade

## Vincoli di game design
- nessun combattimento
- nessun salto
- movimento semplice e leggibile
- interazioni essenziali
- testi brevi
- ritmo lento e contemplativo
- pochi elementi, ma tutti significativi

## Vincoli concettuali
- il gioco non deve banalizzare Foscolo
- il reale meccanicistico deve essere percepibile e disturbante
- il tempio deve essere il centro simbolico del gioco
- il bivio iniziale deve essere una scelta apparente
- i due percorsi iniziali devono convergere
- le illusioni non vanno trattate come menzogne ingenue, ma come costruzioni necessarie

## Priorità tecnica
1. ordine del progetto
2. leggibilità del codice
3. modularità degli script
4. facilità di estensione futura
5. placeholder chiari dove mancano asset finali

## Regole di implementazione
- mantenere script separati per responsabilità
- usare nomi file chiari
- non inserire funzionalità non richieste
- non aggiungere sistemi complessi inutili
- non creare UI invadente
- privilegiare semplicità robusta

## Prima consegna attesa
La prima versione deve includere:
- struttura cartelle
- scene nominate correttamente
- player che si muove
- sistema di interazione base
- cambio scena base
- scena 1 e scena 2 funzionanti con placeholder
- macchina con ingranaggi rotanti anche provvisori

## Cosa evitare
- platform classico
- HUD da action game
- nemici
- raccolte compulsive di oggetti
- testi lunghi o didascalici
- estetica fantasy generica

## Definizione di successo
Il progetto è sulla strada giusta se già nelle prime due scene si percepiscono:
- origine e malinconia
- indifferenza della macchina
- forza simbolica dell’ambiente
- sobrietà elegante del design
